# Video igre kroz decenije

Školski projekat iz web programiranja.

## Opis
Ovo je interaktivni web sajt koji prikazuje razvoj video igara kroz različite decenije.  
Sajt sadrži:
- **Index** – početna stranica sa pregledom decenija i linkovima na ostale strane  (v_index.html)
- **Kviz** – jednostavan kviz vezan za video igre  (v_kviz.html)
- Ostale stranice koje prikazuju pojedinačne decenije  

## Tehnologije
- HTML  
- CSS  
- JavaScript (kod je zakomentarisan jer nismo još radili JS u školi, ali postoji u fajlu i može se koristiti)

## Fajlovi
- `v_index.html` – početna stranica  
- `v_kviz.html` – kviz  
- ostali HTML i CSS fajlovi za prikaz sadržaja i stilizaciju
